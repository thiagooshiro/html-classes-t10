# TripleTen Cohort 35 — Repositório de Estudos Front-end

Este repositório reúne exercícios, exemplos e pequenos projetos desenvolvidos durante o curso de desenvolvimento web da TripleTen, na Cohort 35. A ideia é organizar o aprendizado em módulos e deixar registrado o progresso em HTML, CSS, responsividade, Flexbox, Grid e JavaScript.

## Como este repositório é organizado

A estrutura foi pensada para unir teoria e prática de forma natural. Em grande parte dos módulos, o conteúdo teórico aparece em arquivos de referência e exemplos, enquanto os exercícios aplicam esses conceitos em páginas e componentes reais. Essa abordagem torna o aprendizado mais claro: primeiro se entende a ideia, depois se vê ela sendo usada em cenários próximos do mundo profissional.

## Destaque: Sprint 6

O sexto módulo merece atenção especial. Ele reúne três exercícios de alta qualidade para praticar não só os conceitos da Sprint 6, mas também construir páginas mais complexas com Grid, Flexbox e responsividade, seguindo uma estrutura de requisitos parecida com projetos reais.

## O que você encontra aqui

- Páginas estáticas em HTML e CSS
- Exercícios práticos de layout e estrutura
- Mini projetos com foco em interfaces web
- Exemplos de uso de Flexbox, Grid e media queries
- Materiais de apoio para aulas e revisões
